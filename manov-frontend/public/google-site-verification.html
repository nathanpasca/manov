Google Search Console Verification File

This file is used for Google Search Console verification via the HTML file upload method.

Instructions:

  1. Go to Google Search Console
  2. Add your property (choose "URL prefix" type)
  3. Select "HTML file" verification method
  4. Download the verification file from Google
  5. Replace this placeholder file with Google's file
  6. Ensure the file is served at the root of your domain
  7. Click "Verify" in Google Search Console

Alternatively, use the DNS TXT record method (recommended) or the HTML meta tag method by adding the google-site-verification meta tag to index.html.
